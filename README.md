# AI Student Learning & Performance Analytics

This project is an AI-powered platform that analyzes student learning behavior using chatbot interaction and domain-based coding assessment.

## Features
- AI chatbot that asks students questions
- Domain-based coding assessment (LeetCode style)
- Student performance analytics
- Personalized learning recommendations

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Node.js / Python  
AI: Machine Learning models
